A Biblioteca de Controle de LED foi desenvolvida para facilitar o gerenciamento de um LED em projetos com Arduino, permitindo executar diferentes comportamentos de forma simples, organizada e reutilizável. Com ela, é possível ligar e desligar o LED, mantê-lo ligado por um tempo determinado, fazê-lo piscar continuamente ou ainda piscar por uma quantidade específica de vezes, tudo através de métodos intuitivos. Para utilizá-la, basta adicionar os arquivos da biblioteca ao diretório de bibliotecas do Arduino (geralmente em Documentos/Arduino/libraries), reiniciar a IDE e incluir a biblioteca no código, criando um objeto que recebe como parâmetro o pino onde o LED está conectado. A biblioteca é composta pelos arquivos LED.h e LED.cpp, e é compatível com diferentes placas, como Arduino Uno e ESP32, sendo uma solução prática para quem deseja abstrair o controle de LEDs e manter o código mais limpo e organizado.
